------ampy comands--------
->put command can copy files from your computer to a MicroPython board
ex:
ampy --port /serial/port put test.py //copy a file called test.py from your computer to the root on  of a board filesystem
ex 2:
ampy --port /serial/port put test.py /foo/bar.py //copy a test.py from your computer to a file /foo/bar.py on the board run

 The put command will always overwrite files on the board without warning!
